# Changelog

## 0.1.0 | 17 May 2026

- First release. A fork of BepInEx's Configuration Manager with the F1 settings menu translated into 11 languages.
- Based on Configuration Manager 18.4 by MarC0 / ManlyMarco. Source: github.com/BepInEx/BepInEx.ConfigurationManager.
- Do not install alongside upstream Configuration Manager. If both are present, this version detects the conflict and disables itself, leaving upstream to handle the F1 menu.
