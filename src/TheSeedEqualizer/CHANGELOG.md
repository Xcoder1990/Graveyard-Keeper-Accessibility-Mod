# Changelog

## 1.3.13 | 17 May 2026

- Fixed a launch crash on the 32-bit GOG build with certain mod combinations
- Mod settings menu (F1) labels now follow your in-game language

## 1.3.12 | 3 May 2026

- Settings changes now take effect immediately without restarting the game
- New seed ledger tracks seeds spent vs returned per crop. Toggle off under Tracking.
- Improved diagnostic logging for bug reports

## 1.3.11 | 27 April 2026

- Fixed Chinese translations not loading
- Added a main-menu notice when a newer version is on Nexus. Toggle off in settings if you don't want it
- The Advanced section's Debug Logging option is now always visible (was hidden by default)
- Settings menu reorganised. Existing values preserved.
- Improved debug logging for easier bug reports
- When debug logging is on, a one-time on-screen reminder appears so you don't leave it on by accident
- Added translations for the debug reminder across all 11 supported languages

## 1.3.10 | 11 April 2026

- Added startup logging to help diagnose mod and game environment issues

## 1.3.9

- Fixed player garden seed output not applying until game restart after enabling in config
- GYK Helper is no longer required

## 1

- Initial release
